* 쇼핑몰 웹사이트 제작
  - 백엔드 개발자를 꿈꾸는 사람으로써 웹 프로젝트의 경험이 필수라고 생각하여 쇼핑몰 주제를 가지고 프로젝트 진행을 하게 되었다.
  - 흔한 프로젝트인 만큼 기초라고 생각하며 충분히 복잡하고 퀄리티 좋은 웹프로젝트를 만들기에 좋은 주제라고 생각한다.
